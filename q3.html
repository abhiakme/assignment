What is HTML used for?

  1. HTML (Hypertext Markup Language) is the code that is used to structure a web page and its content.
  2. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables.
  3. It is used to design web pages using a markup language.
  4. HTML is the combination of Hypertext and Markup language. Hypertext defines the link between the web pages
  5. A markup language is used to define the text document within tag which defines the structure of web pages.
  6. Some important points
    • HTML is Easy to Learn and Use.
    • HTML is supported by all Browsers.
    • HTML is Basic of all Programming Languages,etc.

What is CSS used for?

  1. CSS helps us to control the text color, font style, the spacing between paragraphs, sizing of columns, layout designs, and many more.
  2. It is independent of HTML, and we can use it with any XML-based markup language.
  3. Css is easier to maintain and update,Greater consistency in design,More formatting options ,etc

    4. There are three types of CSS which are given below:

    1. Inline CSS.
    2. Internal or Embedded CSS.
    3. External CSS.
